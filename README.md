## Project Overview

This project is for our 4th semester Aerospace Engineering course.

**Title:** ROS-Enabled Design and Implementation of an Intelligent Autonomous Aeroplane  

**Goal:** Build a small-scale autonomous VTOL RC plane using **Pixhawk + Raspberry Pi**, capable of:
- Automatic takeoff and landing  
- Stable flight and waypoint autopilot  
- Live HD video streaming to a ground station  
- Basic obstacle detection and avoidance  
- Weekly documented progress on GitHub (code + tests + reports)
